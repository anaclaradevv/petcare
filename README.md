Descrição do Projeto PetCare: 

É um sistema simples e intuitivo para cadastro e controle de pets. Com ele, é possível registrar informações como nome, espécie (gato ou cachorro), vacinas, peso, data da última consulta e até uma foto do animal. Ideal para clínicas veterinárias, pet shops ou tutores que desejam manter a saúde dos seus pets sempre em dia.
